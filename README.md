# 包裹管理系统

吉林大学2024级高级程序语言设计课程设计项目

## 系统概述

这是一个菜鸟驿站管理系统，主要功能包括用户管理、包裹管理、货架管理等。系统采用C语言开发，包含多个功能模块协同工作。

## 主要功能模块

- 用户管理系统
- 包裹管理系统
- 货架管理系统
- 反馈处理系统
- 好友管理系统
- 预测系统
- 对于功能的细节都在文档的课设报告中

## 使用说明
注意使用GBK编码（vs），其他的编译器保证编码正确也可运行
编译并运行main.c文件即可启动系统。

## 项目以外的说明
这个项目最终我们小组四人分别得分98，95，90，80（机考时未拿优）。
关于吉林大学的课设，重要的拿分点还是在答辩的表现，如果说你只想要混一个及格分，那么大可下载网上一些成熟的系统魔改一下，答辩的时候功能甚至逻辑不正确，只要能跑起来，程序不出现大的崩溃，那么及格是没有问题的。
各个老师有各个老师的提问习惯，有的老师只提问功能，更多考虑功能实用性（这样的话系统更花哨可能更易讨喜）。
而有的老师更在乎代码的质量，代码的可读性，代码的可维护性等等。
有的老师喜欢暴力输入，比如到澡堂点一碗电脑吃。
！！对于git使用，建议大家都去学习，在周边同学还在使用微信甚至不会传源文件的时候，使用git既可以大大提高效率也是一个非常大的加分点！！
对于我们的项目：老师主要提出以下几个问题：
1.密码没有使用掩码输入
2.缺少手机号和快递公司的属性
3.系统内有删除系统记录的功能，公安机关来查了怎么办？？？没错老师就是这样问的。
4.货架的预警没有推送到入库界面和普通用户

另外，我们都是初次学习c语言，尽管经过了多次重构，并且使用大量的安全验证，也成功提高了代码的模块化和可维护性和规范性，但是仍然存在一些问题，还请谅解。
